# TZ-without-docker

h1 Данный проект представляет собой выполнение тестового задания от Интернет-агентство IQ Development.
=====================

Само ТЗ: <https://github.com/iqonline/web_development_test>
***********************
Тестовое задание было реализовано в docker среде для удобста работы с проектом и возможности быстро разворачивать проект на других машинах.
(К сожалению не смог нормально на github залить со средой докера, так что прешлось без)
***********************
Среда docker была сделана на основе существуещего [проекта](https://github.com/drandin/docker-php-workspace), но переделана под мои нужды.
***********************

Также я немного переделал логику расчета вклада чтобы она была правильнее (как я понял).

$summn = $summn + ($summn+$summadd)*($daysn*(0.10/$daysy)); 
        $summn = $summn +$summadd; // добавляем ежемесячный вклад

summadd - сумма ежемесячного пополнения вклада если такой имеется
daysn - кол-во дней в месяце
daysy - кол-во дней в году 
summn - вклад/вклад за прошлый месяц

Конечная сумма вклада округляется до двух цифр после точки.
